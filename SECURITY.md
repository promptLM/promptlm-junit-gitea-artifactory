# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

- Prefer using GitHub Security Advisories for private reporting when available.
- If private reporting is not available, open a GitHub issue with minimal details and wait for a maintainer response before sharing sensitive information.

We will acknowledge receipt within 72 hours and provide a timeline for remediation after triage.

## Supported Versions

This project is in early development. Only the latest release is supported with security fixes.
